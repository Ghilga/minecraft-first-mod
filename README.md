# minecraft-first-mod
a simple mod for minecraft made while learning how to mod

Minecraft version: 1.16.2
Forge version: 33.0.37

## features
- ruby
- ruby ore block
- ruby block
- ruby sword
- ruby furnace recipe
- ruby full armor set
- sapphire (unfinished)
- combat and loot message log in chat
